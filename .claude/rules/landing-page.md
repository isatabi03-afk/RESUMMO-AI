# Landing Page — resummo.com

## Stack
- **Builder:** Lovable (o Next.js si se migra)
- **Dominio:** resummo.com (GoDaddy, S/97/anio)
- **Hosting:** Lovable / Vercel
- **Email corporativo:** @resummo.com (S/3/mes)

## Principios de diseno
- **Mobile-first obligatorio** — 80%+ del trafico viene de Instagram → mobile
- **Velocidad** — Core Web Vitals target: LCP <2.5s, FID <100ms, CLS <0.1
- **Confianza** — social proof visible above the fold (testimonios, badge TOP 2%, contador clientes)
- **Conversion directa** — CTA de compra visible en hero y repetido cada 2 secciones

## Objeciones a manejar en la landing
1. **Precio** → Comparar con academias (S/90 vs S/3,000-5,000)
2. **Desconfianza** → Testimonios reales, fotos de Andre/Nair, badge universidades
3. **No conocen Notion/Anki** → Seccion explicativa con video corto o GIF
4. **Quieren ejemplos** → Preview section con capturas reales por paquete

## SEO basico
- Title: "Resummo — Resumenes de Medicina para ENAM, EsSalud y Ciencias Basicas"
- Meta description: incluir keywords target
- OG tags para compartir en redes (titulo + imagen + descripcion)
- Alt text en todas las imagenes
- URL amigables (resummo.com/paquetes, resummo.com/faq)

## Integraciones en la landing
- **Meta Pixel** — en todas las paginas (ViewContent, AddToCart, InitiateCheckout, Purchase)
- **GA4** — measurement ID, eventos de conversion
- **Pasarela de pagos** — boton de compra redirige a checkout de EasyPay/Culqi
- **WhatsApp widget** — numero de contacto de Resummo
