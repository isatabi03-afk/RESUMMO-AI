# Docs — Resummo

## Estructura

```
docs/
├── discovery/        Transcripciones de reuniones con Andre y Nair
├── arquitectura/     Diagramas, decisiones tecnicas, test results
├── paquetes/         Spec de cada paquete (contenido, precio, Notion page ID)
└── roadmap/          Roadmap por fases, timeline, dependencias
```

## Discovery
Guardar aqui transcripciones y notas de reuniones con Andre/Nair.
Naming: `[YYYY-MM-DD]_[tema].md`

## Arquitectura
Decisiones tecnicas documentadas con razonamiento.
Naming: `decision_[tema].md` o `test-results-[fecha].md`

## Paquetes
Un archivo por paquete con spec completa:
- `ciencias-basicas.md`
- `essalud-enam.md`
- `all-access.md`
- `encaps-anki.md`

Cada spec incluye: nombre, precio, que incluye, Notion page ID, Drive link (si aplica).

## Roadmap
- `roadmap-general.md` — Las 4 fases con timeline
- `fase-1-detalle.md` — Checklist detallado de Fase 1
